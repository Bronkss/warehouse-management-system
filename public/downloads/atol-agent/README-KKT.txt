НАСТРОЙКА ЛОКАЛЬНОЙ ККТ АТОЛ ДЛЯ СКЛАДСКОЙ СИСТЕМЫ
======================================================

Рабочая схема:
- POS в браузере обращается к локальному агенту: http://127.0.0.1:3108
- Токен агента: my_secret
- Агент запускается на ПК кассы через Node.js.
- Агент общается с АТОЛ Драйвером 10.10 через COM AddIn.Fptr10 и PowerShell bridge.
- Маркировка проверяется до добавления товара в чек и повторно перед фискализацией.
- Если результат маркировки не [M+], чек не должен пробиваться.

1. ЧТО УСТАНОВИТЬ НА ПК КАССЫ
-----------------------------
1) Установить АТОЛ Драйвер ККТ 10.10.8 под разрядность Windows:
   - KKT10-10.10.8.0-windows64-setup.exe для 64-bit Windows
   - KKT10-10.10.8.0-windows32-setup.exe для 32-bit Windows

2) Установить Node.js LTS.

3) В утилите АТОЛ Драйвера:
   - выбрать вашу ККТ;
   - проверить связь;
   - сохранить настройки подключения;
   - убедиться, что драйвер видит ККТ.

2. КУДА ПОЛОЖИТЬ ФАЙЛЫ АГЕНТА
-----------------------------
Создайте папку, например:
D:\warehouse-management-system\app\atol-local-agent-driver

Внутри должна быть структура:
server.js
package.json
.env
start-agent.bat
bridge\atol-json.ps1

Файл .env нужно скачать из окна "Настройки ККТ" в системе, чтобы POS_ORIGIN был правильным для вашего адреса сайта.

3. КАК ЗАПУСКАТЬ
----------------
1) Откройте папку агента.
2) Запустите start-agent.bat двойным кликом.
3) Не закрывайте окно консоли, пока касса работает.
4) В консоли должно быть:
   ATOL driver local agent started: http://127.0.0.1:3108
   Mode: driver COM / PowerShell bridge

4. КАК НАСТРОИТЬ БРАУЗЕР КАССЫ
------------------------------
В POS откройте "Настройки ККТ" и сохраните:
Адрес: http://127.0.0.1:3108
Токен: my_secret

Затем нажмите "Проверить связь".

5. ЕСЛИ НЕ РАБОТАЕТ
-------------------
- Проверьте, что start-agent.bat запущен.
- Проверьте, что АТОЛ Драйвер видит ККТ.
- Проверьте, что порт 3108 не занят другим процессом.
- Проверьте, что в браузере сохранён токен my_secret.
- Если зависли процессы Node/PowerShell, выполните:
  taskkill /F /IM node.exe
  taskkill /F /IM powershell.exe
  затем запустите start-agent.bat снова.
